CreateDateFolder
===============
![MIT licensed][shield-license]

概要
-----------------
 * CreateDateFolderは右クリックメニューに「日付フォルダの作成」を追加し、現在の日付（YYYYMMDD)フォルダを作成します。  
 設定画面で、通し番号の付与、日付と通し番号の間の区切り文字を設定することができます。
